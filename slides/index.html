How do computers and computer programs work?

(At a very VERY high level. We won't get into the details.)
  • CPU - reads in instructions that are in the form of 0's and 1's. For example, 010100010101 (I made this up) might mean add two numbers and store it in a register.
  • Program (executable)- A file that is a list of instructions.
  • Code - Human-readable text that gets translated by a compiler/interpreter to 0's and 1's.

Variable assignment/reassignment

Let: x = 5

Now, let's say: x = x + 1

What does this mean?

The = sign means assignment. We first take a look at the right side of the = sign.

Let: x = 5

x = x + 1

Now substitute the variables on the right side


x = 5 + 1


x = 6

So, x = x + 1 means that we are increasing the value of the variable x by one. The right side is the old value and the left side is the new value.


Can you think of why we would need to change the value of a variable?

Some Variable Types

  • Numbers
  • Booleans - true or false
  • Strings - "hello", "asdfasdf123". A string of letters/symbols/numbers. Denoted by the quotation marks ""
    • Tip: "123" is NOT the same as 123

Javascript?

In Javascript, we create functions like this: 


function f(x){
    // We will return the value 1 greater than the given argument
    return x + 1;
}

This is the equivalent of saying: f(x) = x + 1

If you want the computer to ignore a line you typed, prepend it with a //. This is called a comment.

Curly braces show the beginning/end of a function, and semicolons show the end of a statement. Sort of.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );
